# ai-toolbox 🦾

峰哥的个人 AI 工具集。

## 目录结构

- **/skills**: 存放 Nanobot 或 MCP 相关的技能定义与实现。
  - `skill-creator`: 官方技能的本地克隆版，方便自定义修改。
- **/scripts**: 存放各种 AI 相关的独立运行脚本。
- **/utils**: 存放通用的工具函数和公共库。
- **/references**: 本地参考资料，作为 AI 处理任务时的“真值源”。
- **/prompts**: 经过测试的高质量 Prompt 模板。
- **/tests**: 自动化测试脚本，确保工具稳定性。

## 开发指南

1. 保持代码整洁，模块化。
2. 每一个重要的工具都建议配上简单的 README 说明。
3. 提交代码走 PR 流程，方便 Review。
